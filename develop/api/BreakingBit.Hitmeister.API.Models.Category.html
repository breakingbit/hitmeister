Remarks

On the Hitmeister site, every item must belong to a category. Hitmeister has a multi-level category tree with over 5,000 categories in it, but items can only be in leaf categories in the tree. At the top level of the tree we have the root category, which is guaranteed to have an Identifier = 1.
