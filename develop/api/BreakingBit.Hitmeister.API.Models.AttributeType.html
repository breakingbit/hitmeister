Enum AttributeType

The type of value that the Attribute accepts.

Namespace: BreakingBit.Hitmeister.API.Models
Assembly: BreakingBit.Hitmeister.API.dll
Syntax
[JsonConverter(typeof (StringEnumConverter))]
public enum AttributeType
Remarks

Most Attributes accept free text for their values, but some types of attributes have formatting restrictions.

Any attribute type that starts with "Si_" is a numeric value in the International System of Units. When you submit a value for these attributes, it must contain a number and a unit suffix (e.g. Si_Meter could have any of the following values: 1m; 2,5m; 200.432,43m; 250mm; 7km).
